# Density-of-States-for-FHI-AIMS
Python program to plot the Density of States (DOS) from FHI-AIMS outputs 

# Needed Files
The necessary input files are:
  1. geometry.in
  2. KS_DOS_total.dat
  3. atom_proj_dos_*.dat or atom_proj_dos_spin_*.dat
